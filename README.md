# 🌍 Universal AI Translator

An advanced AI-powered translation platform designed to break language barriers with real-time translations, tone customization, and a sleek modern interface.  
Powered by **OpenAI GPT-4o-mini**, it enables seamless communication across cultures with professional polish.

---

## ⚠️ The Problem
Traditional translators often provide literal translations that miss tone, context, or cultural nuance.  
Users struggle to adapt messages for different audiences — whether casual, professional, or creative.

---

## 💡 The Solution
**Universal AI Translator** goes beyond word-for-word conversion.  
It detects input language automatically, translates into the target language, and allows tone customization (casual, formal, professional, humorous, poetic) to match the context perfectly.

---

## 🔑 Key Features
- **Auto Language Detection** – Instantly identifies the source language.  
- **Tone Customization** – Choose from Casual, Formal, Professional, Humorous, or Poetic.  
- **Real-Time Translation** – Processes input after 1.2 seconds of inactivity.  
- **Translation History** – View, copy, share, or delete past translations.  
- **Modern UI/UX** – Dark-themed interface with smooth interactions.  
- **Powered by GPT-4o-mini** – Context-aware translations with cultural sensitivity.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, TailwindCSS  
- **Backend:** Node.js (optional for logging)  
- **AI Engine:** OpenAI GPT-4o-mini API  
- **Deployment:** Vercel  
- **Storage:** Local translation history with copy/share/delete actions

---

## 🚀 How It Works
1. Enter or paste text (up to 5,000 characters).  
2. Select target language and tone.  
3. Click **Translate Now** or wait for auto-processing.  
4. Review translation output and save it to history.  
5. Manage past translations with copy/share/delete options.
